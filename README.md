## âš™ï¸ How the Application Works

1. **Home Page (`index.html`)**

   * Acts as the entry point of the application.

2. **Create Quiz (`create.html`)**

   * Users can create quiz questions and options.

3. **Browse Quiz (`browse.html`)**

   * Displays available quizzes to attempt.

4. **Quiz Page (`quiz.html`)**

   * Users answer multiple-choice questions.

5. **Result Page (`result.html`)**

   * Displays the final score instantly.

All quiz logic and navigation are handled using **JavaScript (`app.js`)**.
